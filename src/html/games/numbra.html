NUMBRA

Team Size: 7 Artists, 5 Programmers

Development Time: 8 months (sept.2024–apr.2025)

Project Overview: A 3D action-adventure game with stealth, puzzles, and exploration elements. Designed with a strong artistic direction and smooth gameplay. Built using Unreal Engine 5. I was responsible for:

  • Creating the camera system and related tools
  • Developing dynamic UI
  • Implementing game localization
  • Building environmental events
  • Contributing to design

We created and implemented gameplay systems, tools, and mechanics in Unreal Engine 5.
